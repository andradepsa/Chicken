⚙️ Technical Architecture

PiCoin Extended was developed based on the robust Bitcoin Core architecture, with an ultra-minimal modification in just 1 file:

// Condition for block subsidy payment:
// ( ( (block_timestamp - jan_1_2021_timestamp) / seconds_per_year - 2025 ) / 140 ) < 70
return (((pindexPrev->GetBlockTime() - 1609459200) / 31536000 - 2025) / 140 < 70) ? nSubsidy : 0;

This rigorously minimal approach modifies only 1 line of code, implementing the extended emission period while maintaining 100% compatibility with Bitcoin Core's proven architecture (excluding this emission rule).

❓ Frequently Asked Questions

🤔 Why based on Bitcoin and not Pi Network?

Proven architecture. Bitcoin has over 15 years of flawless operation. We used its robust architecture as the foundation for PiCoin Extended, ensuring maximum reliability.

🔄 How does the extended emission work?

Time-based condition. The system uses a formula based on the previous block's timestamp relative to January 1, 2021. As long as a specific calculation involving this time difference remains below a threshold, the standard Bitcoin block subsidy (subject to halvings) is paid. This effectively extends the period of subsidy payments for many millennia.

🛡️ Does this affect PiCoin's security?

The core consensus algorithms of Bitcoin are maintained. The modification is in the rule determining how long the block subsidy is paid. While this alters the long-term economic incentives for miners, the fundamental security mechanisms of Bitcoin's PoW and transaction validation remain unchanged. The goal is to enhance long-term security by sustaining miner rewards.

⏰ What is the purpose of the constants (140, 70, 2025) in the formula?

Defining emission duration. These constants, in conjunction with a base timestamp (Jan 1, 2021), determine the total length of the block subsidy emission period. The formula `(((BlockTime_Years_Since_2021 - 2025) / 140) < 70)` results in an emission period lasting approximately 11,825 "block years" from 2021, extending block rewards until roughly the year 13,846.

📅 When exactly will the block subsidy end?

Approximately the year 13,846. The emission of new coins via block subsidy is programmed to cease when the time-based condition in the code is no longer met. This is calculated to occur after approximately 11,825 "block years" have passed since January 1, 2021.
